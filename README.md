
# 🚀 Yapay Zeka Destekli Web Tasarım Projesi (Antigravity)

**Geliştiren:** Ömer Çetin Yalman  
**Üniversite:** Kahramanmaraş İstiklal Üniversitesi  
**Bölüm:** Yazılım Mühendisliği  
**Ders:** Web Tasarımı ve Programlama  
**Danışman:** Dr. Öğr. Üyesi İbrahim ARUK

---

## 📝 Proje Hakkında
Bu proje, modern yapay zeka araçlarının (Claude, Cursor, Gemini) bir web tasarım sürecine nasıl entegre edilebileceğini göstermek amacıyla hazırlanmıştır. Proje kapsamında "Antigravity" temalı, kullanıcı dostu ve modern bir arayüz geliştirilmiştir.

### 🛠️ Kullanılan Teknolojiler
* **Diller:** HTML5, CSS3, JavaScript
* **Framework:** Bootstrap 5
* **Yapay Zeka Araçları:** Claude AI, Cursor Editor, Google Gemini

## 📂 Proje İçeriği ve Kurulum
1.  Tüm kod dosyalarına yukarıdaki dosya listesinden erişebilirsiniz.
2.  `index.html` dosyasını tarayıcınızda açarak projeyi canlı olarak inceleyebilirsiniz.
3.  Tasarımlarda **Glassmorphism** efektleri ve **Bootstrap** grid sistemi kullanılmıştır.
